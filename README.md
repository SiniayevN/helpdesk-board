Helpdesk Board

A Next.js App Router project that displays helpdesk tickets with filters, search, queue management, and live updates.

Setup
1. npm install  
2. npm run dev  
3. Visit http://localhost:3000  

Build
1. npm run build  
2. npm start  

Features
* Loads tickets from /api/tickets (JSON)  
* Filter by Status and Priority  
* Search by title or description  
* Add, remove, or clear items in “My Queue”  
* Loading, error, and empty states  
* Live updates every 6–10 seconds  
* Clean dark UI with Tailwind CSS  

Acceptance Criteria
* Tickets load on first visit  
* Filters and search are fully controlled  
* Queue actions and messages work correctly  
* Live updates run with proper cleanup
